## 🎯 QUICK START WORKFLOWS

### Scenario 1: Find Best Material for Aerospace
```
1. Click "💡 Use Cases" tab
2. Select "Aerospace"
3. View 10 recommended materials
4. Click material to see details
5. Use "Compare" tab to evaluate 2 materials
```

### Scenario 2: Find Cheap Strong Material
```
1. Click "🔍 Advanced Search" tab
2. Set Max Cost: $50
3. Set Min Strength: 500
4. Sort by: Cost
5. Browse results
```

### Scenario 3: Compare Two Materials
```
1. Click "⚖️ Compare" tab
2. Select: "Titanium" and "Carbon Fiber"
3. View side-by-side table
4. See radar chart comparison
5. View property breakdown
```

### Scenario 4: Find Lightweight Material
```
1. Click "🎯 Quick Select" tab
2. Set category to "All"
3. Drag Weight slider to max: 2 g/cm³
4. See filtered results
5. Check alternatives
```
